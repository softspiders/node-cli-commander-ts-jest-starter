## Install

```sh
npm i -g
```

## Run

```sh
cli-command
```
or
```sh
cli-command -h
```

Should see:

```sh
Usage: cli-command [options] [command]

Options:
  -V, --version    output the version number
  -h, --help       output usage information

Commands:
  write [message]  say hello!
  help [cmd]       display help for [cmd]
```


```sh
cli-command -V
```

Should see:

```sh
0.1.0
```
